tranfert_local · Projet C++17 / SFML

Dernière mise à jour 2026-04-22

Features livrées

  • 📡 LocalTransfer (noyau)

    Application desktop cross-platform (macOS ↔ Windows) pour transférer fichiers et dossiers en peer-to-peer sur un réseau local, sans cloud, sans configuration. Découverte UDP, transfert TCP, code PIN d'appairage, progression temps réel, multi-destinataires.

    C++17 SFML 2.6 CMake Cross-platform v1.0 · Audit 85/100
  • 🌐 Interface Web embarquée

    Serveur HTTP interne (port 45456) qui expose LocalTransfer à tout navigateur du LAN : upload/download bidirectionnel sans installation, liste de pairs unifiée (natifs + web) côté desktop, auto-propagation du binaire natif selon l'OS (Windows .exe, macOS .app zippé, Linux ELF), QR code et PIN d'accès dans l'app desktop.

    HTTP cpp-httplib SSE Vanilla JS QR code WebRTC DataChannel v1.2 · web↔web P2P · 14/14 tests
